Microblogger

Built with the SQLite3 - 2023

I built a microblogging web application, designed to be a one-stop hub for all my video game blogs. The primary idea was to make data organization a breeze, allowing for easy sorting and filtering through a database. In terms of design and styling, I took the opportunity to challenge and improve my UI skills. I stuck with vanilla CSS, striving to create an engaging and user-friendly interface for my visitors, who are all about exploring video game-related content. It was all about enhancing the user experience while keeping things simple and effective.

Role and responsibility

As the developer and creator of the microblogging web application, I was responsible for the following:

1. Conceptualization and Design: I initiated the project, outlining the concept and core functionalities. I handled the application's design, ensuring an appealing and user-friendly interface that aligned with the video game feel. It should feel like a video game website.

2. Front-End Development: My responsibilities included the development of the front-end components using HTML and CSS. I meticulously crafted the layout, styling, and responsive design to provide an engaging user experience.

3. Back-End Development: I designed and developed the back-end of the application, utilizing SQLite3 for database management. This involved setting up the database schema, user authentication, and ensuring the seamless flow of data between the front end and back end.

4. Content Sharing and Interaction: I integrated features for users to like posts, and leave comments. Creating a robust and interactive platform was a key objective.

The design

The design of my microblogging web application was thoughtfully crafted to resonate with a video game-inspired aesthetic. The aim was to create a visual experience that resonated with the interests of the target audience, providing a sense of immersion into the world of video gaming. I also wanted it to come across immediately as a blog website, so in keeping with the video game aesthetic I made sure it was minimal and mimicked other blogging websites.

In line with the gaming theme, the design was intended to be both visually engaging and easy to navigate. I took inspiration from the visual elements found in video games, incorporating graphics, color schemes, and layouts that were reminiscent of the gaming world; blues and purples.
